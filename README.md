# 💎 Aoro Joias - Site Institucional

Projeto interdisciplinar desenvolvido para as disciplinas de **Comunicação e Expressão** e **Informática para Negócios** da **Fatec Rio Preto**, no curso de **Informática para Negócios**.

---

## 📌 Sobre o Projeto

Este projeto tem como objetivo o desenvolvimento de um **site institucional para a Aoro Joias**, uma loja real do segmento de joias.

A proposta integra conceitos de comunicação, design e desenvolvimento web, visando criar uma experiência moderna, elegante e funcional para os usuários.

---

## 🎯 Objetivos

* Desenvolver um site institucional profissional
* Aplicar conceitos de design e comunicação visual
* Trabalhar a experiência do usuário (UX)
* Integrar conhecimentos técnicos e teóricos das disciplinas
* Simular um ambiente real de desenvolvimento para um cliente

---

## 🛠️ Tecnologias Utilizadas

* HTML5

* JavaScript
